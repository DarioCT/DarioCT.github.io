ISIC Challenge on Skin Lesion Analysis Towards Melanoma Detection¶

In this project, we will design an algorithm that can visually diagnose melanoma, the deadliest form of skin cancer. In particular, the algorithm will distinguish this malignant skin tumor from two types of benign lesions (nevi and seborrheic keratoses).

Skin cancer is a major public health problem, with over 5 million newly diagnosed cases in the United States each year. Melanoma is the deadliest form of skin cancer, responsible for over 9,000 deaths each year.

The data and objective are pulled from the 2017 ISIC Challenge on Skin Lesion Analysis Towards Melanoma Detection. As part of the challenge, participants were tasked to design an algorithm to diagnose skin lesion images as one of three different skin diseases (melanoma, nevus, or seborrheic keratosis).

In this project, we will create a convolutional neural network model based on transfer learning for the classification problem at hand.

Image Data¶

The data consist of 2000 training images, 150 validation images and 600 testing images. All images are HD quality.

Images belong to three separate categories:

Melanoma  malignant skin tumor, derived from melanocytes (melanocytic)
Nevus  benign skin tumor, derived from melanocytes (melanocytic)
Seborrheic keratosis  benign skin tumor, derived from keratinocytes (non-melanocytic)

There are 2000 images provided as training data, including 374 "melanoma", 1372 "benign nevi", and the remainder 254 are "seborrheic keratosis".

Data Pipline¶

The training data pipeline is going to perform following tasks:

1. Image and target file paths import
2. Targets conversion to binary class matrix
3. Image resize to 299x299
4. Image scale by factor 1./255
5. Feature extraction using InceptionResNetV2 model
6. Extracted features are shuffled and fed into the top CNN model

The validation and testing data pipeline is going to perform following steps:

1. Image and target file paths import
2. Targets conversion to binary class matrix
3. Image resize to 299x299
4. Image scale by factor 1./255
5. Feature extraction using InceptionResNetV2 model

def path_to_tensor(img_path):
    import cv2
    # loads BGR image
    img = cv2.imread(img_path)
    # convert image to 3D tensor with shape (299, 299, 3)
    img = (cv2.resize(img, (299, 299)).astype(float))/255.0
    # convert 3D tensor to 4D tensor with shape (1, 299, 299, 3) and return 4D tensor
    return np.expand_dims(img, axis=0).astype(float)

def paths_to_tensor(img_paths):

Transfer Learning Model¶

As previously mentioned we are using transfer learning method to create our model. Our model will use the pretrained deep convolutional neural network called InceptionResNetV2. We are going to create a small network that will take the output of the InceptionResNetV2 model (without its top fully connected layers). The InceptionResNetV2 model has 55,873,736 parameters and 572 hidden layer and has been pretrained on ImageNet data. The schematic of the InceptionResNetV2 model can be seen below.

Ref: https://ai.googleblog.com/2016/08/improving-inception-and-image.html

On top of InceptionResNetV2 model there is smaller model starting with GlobalAveragePooling2D layer followed by two fully connected layers after each is a dropout layer and a three category classification output using softmax as the final layer. The network summary can be seen below:

The model was trained using Adam optimizer and categorical crossentropy loss as the variable for minimization. Learning rate decay was used to achieve better training by reducing the learning rate by a third if the validation loss did not improve in the last 9 epochs. After each batch the train loss and validation loss as well as training accuracy and validation accuracy have been printed. The model was set to train from 200 epochs. Early stopping was used to stop the model training if the validation loss did not improve after 35 epochs. Model weights were saved after each decrease in validation loss.

Task 1: ROC AUC for Melanoma Classification¶

In the first task, we will gauge the ability of our CNN model to distinguish between malignant melanoma and the benign skin lesions (nevus, seborrheic keratosis) by calculating the area under the receiver operating characteristic curve (ROC AUC) corresponding to this binary classification task.

A receiver operating characteristic (ROC) is a graph that illustrates the performance of a binary classifier as its discrimination threshold (cutoff) is changed. The curve is created by plotting the true positive rate (TPR) against the false positive rate (FPR) at various cutoff settings. The true-positive rate is known as sensitivity, the false-positive rate is known as the fall-out and is calculated as (1 - specificity). This means that the top left corner of the plot is the ideal point - a false positive rate of zero, and a true positive rate of one. Larger area under the curve (AUC) indicates better score.

To solve this task a new function was created to plot ROC curve and calculate AUC score.

Ref: https://kennis-research.shinyapps.io/ROC-Curves/

In [17]:
from sklearn.metrics import roc_curve
from sklearn.metrics import auc

def plot_roc_auc(y_true, y_pred):

    # initialize dictionaries and array
    fpr = dict()
    tpr = dict()
    roc_auc = np.zeros(3)

    # prepare for figure
    plt.figure()


    # obtain ROC curve
    fpr, tpr, _ = roc_curve(y_true, y_pred)
    # obtain ROC AUC
    roc_auc = auc(fpr, tpr)
    # plot ROC curve
    plt.plot(fpr, tpr, lw=2, label='ROC curve (area = {:.2f})'.format(roc_auc))


    # format figure
    plt.plot([0, 1], [0, 1], 'k--', lw=2)
    plt.xlim([0.0, 1.0])
    plt.ylim([0.0, 1.05])
    plt.xlabel('False Positive Rate')
    plt.ylabel('True Positive Rate')
    plt.title('ROC curve')
    plt.legend(loc="lower right")
    plt.show()
